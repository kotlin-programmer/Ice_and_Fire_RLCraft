Les cristaux d'invocation de dragon 
peuvent être fabriqués pour se lier
à un dragon, ce qui permet de le 






téléporter à l'endroit où vous 
vous trouvez.
Lorsqu'il est utilisé sur un dragon, 
le cristal se lie à lui.
En cliquant à nouveau sur le cristal
dans la même dimension, le dragon 
sera téléporté.

Pour délier le cristal, il suffit de le 
placer seul dans la grille de 
fabrication.
